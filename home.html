    Help & Support

    Connect Devices (Proxy)

    After joining this hotspot set a Manual HTTP Proxy so browsing is unlocked after ads. Use these settings:

    • Host / Server: loading...
    • Port: ...
    • PAC (auto): loading...
    • If PAC used, choose "Auto" in Wi‑Fi proxy settings.

    🔧 Important: You MUST configure proxy settings to access internet after watching videos. Without proxy configuration, only this portal will be accessible.

    Android steps Open Wi‑Fi network details > Edit > Advanced > Proxy: Manual > Host & Port above. Or select Auto and paste PAC URL.
    Windows steps Settings > Network & Internet > Proxy > Manual proxy setup > On. Enter Address & Port above. Or use Script address with PAC URL.
    iPhone / iPad steps Settings > Wi‑Fi > (i) on network > Configure Proxy > Manual (or Automatic) then enter Host, Port or PAC URL.
    Without the proxy only this portal & limited media domains are reachable.
